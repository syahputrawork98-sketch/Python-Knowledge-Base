# Sub-Rak 05: Ecosystem & Tooling (Dunia Nyata Pengembangan)

Fase inti terakhir ini bukan tentang struktur bahasa, melainkan *Bagaimana cara menulis perangkat lunak Python ala profesional*?

## Daftar Buku

1. [**01_standard_library**](01_standard_library/)
   Eksplorasi pustaka standar bawaan (Baterries Included) seperti `collections`, `itertools`, `datetime`, dan `typing`.

2. [**02_testing**](02_testing/)
   Memastikan kode berjalan seperti yang diharapkan. Dari `unittest` bawaan hingga `pytest` modern.

3. [**03_packaging**](03_packaging/)
   Bagaimana mengubah kumpulan file `.py` menjadi paket yang bisa didistribusikan via PyPI (pip).

4. [**04_virtual_environments**](04_virtual_environments/)
   Isolasi proyek (`venv`, `virtualenv`), serta alat manajemen defacto modern lainnya.

5. [**05_tooling**](05_tooling/)
   Lingkungan kerja produktif: Linter (flake8/ruff), Formatter (black), Type Checker (mypy), dan CI/CD.
