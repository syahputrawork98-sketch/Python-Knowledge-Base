# Sub-Rak 03: System & IO (Interaksi dengan Sistem)

Fase ini memandu Anda keluar dari batas "ruang bahasa Python" murni, menuju bagaimana kode Python berkomunikasi dengan sistem operasi, file, jaringan, dan lingkungan di luarnya.

## Daftar Buku

1. [**01_modules_import_system**](01_modules_import_system/)
   Bagaimana Python menemukan, memuat, dan mengelola direktori, modul, serta *packages*. Konsep `sys.path`, ruang lingkup import, dll.

2. [**02_file_system_io**](02_file_system_io/)
   Operasi baca-tulis file, manipulasi path (seperti `os` dan `pathlib`), dan menangani berbagai format file (JSON, CSV, dll).

3. [**03_networking**](03_networking/)
   Komunikasi melalui protokol web (HTTP/s), Socket programming, memanggil API dengan `requests` atau `urllib`.
