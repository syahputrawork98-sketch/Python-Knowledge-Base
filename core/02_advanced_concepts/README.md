# Sub-Rak 02: Advanced Concepts (Konsep Lanjutan & Di Balik Layar)

Setelah memahami bagaimana menulis Python yang *Pythonic*, sub-rak ini membawa kita menyelam lebih dalam untuk melihat bagaimana kode tersebut diproses, diterjemahkan, dan dikelola oleh penerjemah bahasa itu sendiri.

Fase ini mengubah pembaca dari pengguna Python menjadi pemrogram yang benar-benar memahami mesinnya.

## Daftar Buku

1. [**01_execution_model**](01_execution_model/)
   Cara kerja interpreter, bytecode, kompilasi CPython, *Global Interpreter Lock (GIL)*, scope, dan namespace.

2. [**02_object_model**](02_object_model/)
   Segalanya di Python adalah objek. Apa artinya? Memahami Dunder methods (`__init__`, `__new__`), Metaclasses, *Descriptors*, dll.

3. [**03_memory_management**](03_memory_management/)
   Bagaimana Python mengelola memori? Membahas *Reference Counting*, *Garbage Collection*, konsep *Mutable* vs *Immutable* dari kacamata alokasi memori.
