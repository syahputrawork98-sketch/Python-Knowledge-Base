# Rak 13: Python for Data Science & AI

Tingkatan spesialisasi di mana Python digunakan sebagai pusat kecerdasan data dan rekayasa AI.

## Struktur Sub-Rak
- **[SR-01-DATA-CORE](./SR-01-DATA-CORE/README.md)**: Fondasi manipulasi data (NumPy, Pandas).
- **[SR-02-MACHINE-LEARNING](./SR-02-MACHINE-LEARNING/README.md)**: Implementasi algoritma ML (Scikit-learn).
- **[SR-03-AI-ENGINEERING](./SR-03-AI-ENGINEERING/README.md)**: Deep Learning & LLM (PyTorch, LangChain).

## The Pythonic Advantage in AI
Python menjadi bahasa utama AI bukan karena kecepatannya (C++ lebih cepat), tetapi karena:
1. **C-Interoperability**: Library AI Python sebenarnya ditulis dalam C/C++ (High performance behind the scenes).
2. **Readability**: Memungkinkan ilmuwan fokus pada matematika, bukan manajemen memori.
3. **Ecosystem**: Komunitas terbesar untuk riset AI di dunia.
