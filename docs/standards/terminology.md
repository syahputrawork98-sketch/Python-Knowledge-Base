# Panduan Terminologi (Python Edition)

Menekankan keterbacaan dan keanggunan gaya "Zen".

## 1. Aturan Penulisan Istilah
- **Zen-Style**: Gunakan kalimat yang mengalir dan elegan (The Zen of Python).
- **Batteries Included**: Tekankan kemudahan penggunaan library standar.

## 2. Senior vs Basic Terms

| Basic Term | Senior Terminology | Konteks |
| :--- | :--- | :--- |
| Ubah fungsi | **Decorators** | Meta-programming sederhana. |
| Loop ringkas | **List Comprehension** | Cara Pythonic memproses list. |
| Urutan warisan | **MRO (Method Resolution Order)** | Pencarian atribut di multiple inheritance. |
| Kunci global | **GIL (Global Interpreter Lock)** | Batasan threading di CPython. |

## 3. Metode Analogi
- **The Scholar's Library**: Anggap Python sebagai perpustakaan besar di mana setiap rak sudah siap melayani kebutuhan Anda.
- **Satu Jalan Terang**: Selalu arahkan pembaca ke cara "The Pythonic Way".
