# Contribution Guide: Core & External

Dokumen ini mengatur standar kontribusi untuk menjaga integritas teknis Python Knowledge Base.

---

## 🛠️ 1. Standar Kontribusi Teknis

### Digital Mirroring
Setiap penambahan materi baru wajib memiliki referensi 1:1 ke sumber primer (docs.python.org / PEP).

### 8-Point Gold Standard
Setiap unit materi (README.md) di level Bab/Section **WAJIB** memenuhi kriteria PPM V4 yang didefinisikan di `content-workflow.md`.

---

## 🏗️ 2. Alur Kontribusi

1. **Ide/Issue**: Ajukan melalui diskusi Mode `DISCUSS`.
2. **Drafting**: Gunakan Mode `DRAFT` untuk menyusun narasi sebelum memodifikasi file.
3. **Validation**: Gunakan Mode `REVIEW` untuk memastikan kepatuhan terhadap standar visual dan teknis.
