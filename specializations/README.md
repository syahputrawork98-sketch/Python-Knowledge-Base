# Rak Utama: Python Specializations

Rak ini berisi penerapan praktis bahasa Python di berbagai domain industri.

Pastikan Anda sudah memiliki pemahaman yang cukup kuat dari rak [**Core Python**](../core/README.md) sebelum menyelami buku-buku spesifik di sini.

## Daftar Spesialisasi & Karir (Sub-Rak):

1. **[01_web_development](01_web_development/)** - Membangun backend aplikasi web dan REST API.
2. **[02_data_science](02_data_science/)** - Analisis, manipulasi, dan visualisasi data.
3. **[03_machine_learning](03_machine_learning/)** - Algoritma dan pemodelan prediktif.
4. **[04_ai_engineering](04_ai_engineering/)** - Membangun sistem gen-AI, LLM integration, dan RAG.
5. **[05_automation](05_automation/)** - Automasi UI, web scraping, dan scripting pekerjaan berulang.
