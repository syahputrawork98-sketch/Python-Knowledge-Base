# Rak 05: Standard Library (Senjata Bawaan)

> **"Before installing a package, always check if Python already has it."**

Rak ini adalah **Eksplorasi Arsenal Bawaan Python** — modul-modul yang hadir tanpa perlu `pip install`. Menguasai standard library adalah tanda *mastery* yang sesungguhnya.

## 🏗️ Struktur Sub-Rak

| Sub-Rak | Fokus | Modul Utama |
| :--- | :--- | :--- |
| [SR-01-builtins](./SR-01-builtins/) | Built-in Functions | `map`, `filter`, `zip`, `sorted`, `enumerate` |
| [SR-02-functools](./SR-02-functools/) | Functional Programming | `partial`, `lru_cache`, `reduce`, `wraps` |
| [SR-03-collections](./SR-03-collections/) | Data Structures Tingkat Lanjut | `defaultdict`, `Counter`, `deque`, `namedtuple` |

| [SR-05-io-and-path](./SR-05-io-and-path/) | File & I/O | `pathlib`, `io`, `os`, `shutil` |
| [SR-06-concurrency](./SR-06-concurrency/) | Concurrency | `threading`, `multiprocessing`, `asyncio` |
| [SR-07-typing](./SR-07-typing/) | Type System | `TypeVar`, `Protocol`, `dataclass`, `NamedTuple` |

## 🎯 Key Goals
- Mengetahui modul yang tepat untuk setiap kebutuhan tanpa redundansi.
- Memahami perbedaan threading vs multiprocessing vs asyncio.
- Memanfaatkan `functools` dan `itertools` untuk kode yang lebih efisien.
