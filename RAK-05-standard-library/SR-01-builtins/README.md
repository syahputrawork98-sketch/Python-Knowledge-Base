# SR-01: Builtins Mastery (The Core) [x] Complete

> **"Mastering Python's built-in functions is the first step to writing idiomatic, high-performance code."**

Sub-Rak ini mengeksplorasi arsenal fungsi bawaan (**Builtins**) yang paling kritis. Kita akan membedah bagaimana transformasi fungsional dan agregasi data dilakukan di level C melalui fungsi-fungsi seperti `map`, `filter`, `zip`, dan `sorted`, memberikan Anda kekuatan untuk memproses data dengan sangat efisien tanpa library eksternal.

## 🎯 Key Learning Goals
- Memahami beda antara **Lazy Evaluation** (Iterators) vs **Eager Evaluation** (Lists).
- Mampu menggunakan `map` dan `filter` untuk transformasi data fungsional.
- Menguasai teknik agregasi silang menggunakan `zip` dan `enumerate`.
- Memahami cara kerja pengurutan data tingkat lanjut menggunakan `key` di `sorted`.
- Mampu melakukan benchmarking performa menggunakan `timeit`.
