## 🎯 Key Learning Goals
- Memahami motivasi di balik penghapusan ketergantungan pada `%` dan `.format()`.
- Mampu mengimplementasikan operator `:=` untuk menyederhanakan alur kondisional.
- Menguasai teknik **Structural Pattern Matching** untuk memproses data kompleks secara deklaratif.
- Mengetahui batas-batas kompatibilitas fitur antar-versi Python.
