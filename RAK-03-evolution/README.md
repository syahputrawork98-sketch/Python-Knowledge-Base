# Rak 03: Evolution (PEPs & Versi)

> **"Understanding why a feature was added tells you more than knowing how it works."**

Rak ini menelusuri **jejak evolusi Python** melalui PEP (*Python Enhancement Proposal*). Setiap fitur modern Python lahir dari PEP — dokumen proposal yang mendokumentasikan motivasi, desain, dan konsekuensi dari perubahan tersebut.

## 🎯 Key Goals
- Memahami *mengapa* setiap fitur diperkenalkan — masalah apa yang dipecahkan.
- Mampu membaca dan menafsirkan dokumen PEP resmi.
- Mengerti trade-off yang dipertimbangkan sebelum fitur diterima.
